PyAlert

@objc public class PyAlert: NSObject

A class representing an alert to be used from the Python API.

@objc public var title: String? The alert's title.

@objc public var message: String? The alert's message.

@objc public static func alertWithTitle(_ title: String?, message: String?) -> PyAlert Initialize a new alert.

  • title: The alert's title.
  • message: The alert's message.

@objc public func show() -> String Show an alert with set parameters.

  • Returns: The title of the pressed action.

  • title: The title of the action.
  • handler: The action's handler.

  • title: The title of the action.
  • handler: The action's handler.

  • title: The title of the action.
